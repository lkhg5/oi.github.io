01. Lesson Overview

Lesson Outline

This lesson is divided into two sections:

  1. Unit testing refresher - You must have learned about testing in the previous course(s). Since the final project will also require you to write the unit test with specified code coverage, the first half of this lesson presents a refresher. After completing the refresher, you will be able to:
    • Write JUnit tests using Mockito framework
    • Add tests in the eCommerce application
  2. Logging - Next, we will learn how to generate and save logs. The prime objective of this lesson is to describe SLF4J logging API and Log4J logging framework for code insight and debugging. By the end of the lesson, you will be able to:
    • Explain the basics of Logging
    • Use the SLF4J library (API) to implement any logging framework
    • Implement Log4J framework in the eCommerce application

In the next lesson, Splunk, we will learn to analyze the logs to derive meaningful insight.
