ToDo Exercise

  1. If you haven't done it already, fork the Udacity project repository to your Github account. The forked one will now become the remote repository for you.

  1. Clone the remote repository to your local machine using terminal (Mac/Linux)/Gitbash(Windows). You will require to download Git before running the following commands :
git clone https://github.com/<YourGithubUserName>/nd035-c4-Security-and-DevOps.git
cd nd035-c4-Security-and-DevOps

Alternatively, you can clone it directly into your IntelliJ IDE.

  1. Make any changes in the repository, and push the changes back to the remote repository.
    Optional: If you were doing the changes in a text editor, such as Visual Studio Code, you could use the following commands to push the changes back to the remote. 
# Verify the remote. The default name of remote is "origin"
git remote -v
# Open the code in IntelliJ or any editor. 
# Make any change, as small as adding one comment to your code
# Add the changes to the "staging" area: New, Modified, and Deleted files
git add -A 
# Commit your changes with a message to the reader of your repo
git commit -m "my comment"
# Push all changes back to the remote
git push
