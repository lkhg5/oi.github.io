Exercise

Task Description:

Environment Setup
  • MySQL server is started and contains the JDND-C3 database. Create it, if not exists.
  • Java 1.8 is installed.
  • Checkout the starter code
  • Import the starter code into the IDE like IntelliJ IDEA CE (Optional).
Exercise Instructions
  • Create a table called member that represents the members of a health club. The table contains the following columns,
Column Description
id Auto Generated Primary key
first_name First name of the member
last_name Last name of the member
age How old is the member?
gender Gender of the member
balance Outstanding balance the member owes to the club

Choose appropriate data types for each column.

  • Write the SQL to create the member table in src/main/resources/db/migration/V01__CreateTable.sql.
  • Follow the steps in Application class.
How to run the code?
In the IDE,
  • Run the Application class.
Using Maven
  • MacOS - ./mvnw exec:java
  • Windows - mvnw exec:java
