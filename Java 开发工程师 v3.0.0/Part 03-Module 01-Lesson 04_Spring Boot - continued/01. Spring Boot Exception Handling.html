01. Spring Boot Exception Handling

Spring Boot Exception Handling

035ND C01 L04 A01 SPRINGBOOT EXCEPTION

035ND C01 L04 A02 EXCEPTION HANDLING

There are five ways we can use to handle Spring Boot exceptions rather than showing default error pages like 404 or 500 which don't give much detail about the exception. The ways are listed below:

  • Creating a custom error page

  • Using @ExceptionHandler

  • Using @ControllerAdvice + @ExceptionHandler

  • Configuring SimpleMappingExceptionResolver class

  • Using Custom HandlerExceptionResolver class.

Let’s learn these one by one in the coming lessons.
